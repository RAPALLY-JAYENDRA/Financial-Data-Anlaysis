# Financial-Data-Anlaysis
Financial Data Analysis Dashboard built using Power BI, DAX, and Power Query. Performed data cleaning, transformation, and star schema modeling on Balance Sheet, Income Statement, and Cash Flow datasets. Created interactive dashboards for revenue, profit, assets, liabilities, and cash flow insights.

# Financial Data Analysis Dashboard 

## Overview

This project is an interactive Financial Analytics Dashboard developed using **Power BI**, **DAX**, and **Power Query**. The dashboard analyzes annual and quarterly financial data including Balance Sheet, Income Statement, and Cash Flow Statement datasets.

The project focuses on financial performance analysis, profitability trends, cash flow monitoring, and company-level KPI insights.

---

# Tools & Technologies Used

* Power BI
* Power Query
* DAX
* Data Modeling
* CSV Datasets

---

# Dataset Information

The project uses:

* Balance Sheet (Annual & Quarterly)
* Income Statement (Annual & Quarterly)
* Cash Flow Statement (Annual & Quarterly)
